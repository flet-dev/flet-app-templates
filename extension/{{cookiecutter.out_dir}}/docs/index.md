## Installation

Add dependency to `pyproject.toml` of your Flet app:

```
dependencies = [
  "{{cookiecutter.project_name}} @ git+<link to GitHub repository>",
  "flet>={{cookiecutter.flet_version}}",
]
```

Build your app:
```
flet build macos -v
```
